# Olist Brazilian E-Commerce Analysis

## Project Overview
End-to-end analysis of the Olist Brazilian E-Commerce dataset using SQL, Python, and Tableau.

## Tools Used
- Google BigQuery (SQL)
- Python
- Tableau
- GitHub

## Project Files
- SQL Analysis Report: `Olist_SQL_Analysis_Report.pdf`
- Python Notebook: `Brazilian_E_commerce_by_Olist.ipynb`
- Python HTML Report: `Brazilian_E_commerce_by_Olist.html`

## Key Insights
- São Paulo generated the highest revenue and order volume.
- Beauty & Health and Watches & Gifts were top revenue-generating categories.
- Customer retention was low.
- Delivery delays negatively impacted review scores.
- Most reviews were positive, with 5-star ratings dominating.

## Business Recommendations
- Improve customer retention through loyalty programs.
- Reduce delivery delays.
- Focus on high-performing product categories.
- Strengthen operations in high-revenue regions.
- Support top-performing sellers.
